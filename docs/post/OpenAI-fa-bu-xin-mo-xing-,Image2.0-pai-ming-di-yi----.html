OpenAI发布新模型,Image2.0排名第一???

前言

就在昨天，我看到群里有人发了一条消息

是的,从图片上来看Google要正式回归中国，可是这不由出现了一些蹊跷，Google这么多年，在之前被逼退中国市场，可是现在却宣布回归中国，并且在上海作为总部的地址，就这样，我去网上查询了一些相关的内容，理论上来说有这样的消息应该是全网都在讨论，可经过我的多天刷视频下来，却没有一条视频于Google回归中国有关系。
后来，有一名热心群友指正了，原话是“这是chatGPT新出的images-2模型生的图，很强，比谷歌大香蕉高200分，断层”，比Google家出的芭娜娜模型还要高，究竟是何方神圣能够做到这样的效果呢？在零度老师的视频当中，我也有了解到Openai发布的新模型，但只是知道了罢，没有去深层的研究，我想我应该去真实的体验一下这个号称断层的模型了

使用

第一时间体验下来，ChatGPT的ui改动有很多，还有就是动画非常的丝滑，并且连图画也有了一个可视化的过程，这个需要自己亲生去体验才能领悟到Openai的良苦用心，但是，Openai需要进化一下自己的提示词库了，我上传了一张图片

我输入的提示词是"把他改成现实的风格 Make the aspect ratio 16:9"，是将他改成了现实的风格并且使用16:9的图片格式

这个是生成出来的效果图，可以看出Openai在模型训练中花了很大的功夫，在细节的处理上很细腻，背景的虚化也更加真实，在官网的介绍中我看到了更多的图片
Belike:

这些都是出自Image2.0模型之手，应该是专门针对文字部分做了训练,并且在模拟人类字迹的方面也很新颖，解决了之前"AI造字"这一问题

它能理解漫画的格式，这一张图片也是在官网的宣传图中我认为比较喜欢的一张了，这应该是当今中最强的图片生成模型了。

结尾

Image 2.0确实是一个能胜任这个排名的好模型，有兴趣的小伙伴可以去体验一下，祝你有好的一天。(/ω＼)
